# §6a — Masterplan staging (the intervention strategy)
_How the analysis assembles into the masterplan. The map itself is Sonya/Carlton's design; this stages the grounded components and the intervention typology. Layers: the built §3–§5 set + `03_PROCESSED/interventions_6a.fgb` (55 proposed nodes)._

## The spine and three zones (from the xeroriparian strategy)
The **Barranco de Valcuerna** is the ecological spine to reactivate. The strategy's three zones map onto our layers:
1. **Left Ravine** — the barranco corridor itself (4.1 Valcuerna spine + 4.2 gypsum badlands). Xeroriparian reactivation, check dams, erosion control.
2. **Agricultural Crossing** — where the corridor + cañadas cross the intensive irrigated matrix (4.5/4.6 hostile matrix + 4.x cañadas). Hedgerows, field-corner planting, fauna crossings.
3. **River Interface** — the Cinca/Ebro edge (4.1 main rivers + riparian). Riparian stepping-stones, bank cover.

## Grounded intervention nodes (`interventions_6a.fgb`, 55)
- **14 fauna crossings (priority)** at the open barrier × corridor points (from 4.4 — 14 of 16 crossings have no structure). These are the pinch points: motorway/HS-rail/main-road/canal cuts. 2 more flagged "verify" (existing structure).
- **39 hedgerow / field-corner nodes** where cañadas cross intensive farmland (from 4.x-b readiness). These stitch cover across the hostile matrix using public land (no expropriation).
